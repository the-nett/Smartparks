Acerca del Proyecto

El proyecto SmartParks busca monitorear los datos generados por sensores ubicados en parques urbanos, para asi contrinuir con el desarrollo de una ciudad inteligente.

Recolección de Datos

Los sensores recolectan datos como temperatura y cantidad de personas presentes.

Almacenamiento

Esta información es almacenada en una base de datos relacional para su posterior análisis.

Análisis

Procesamiento de datos para generar insights y métricas relevantes.

Toma de Decisiones

Facilitamos la gestión eficiente de espacios públicos basada en datos.

Objetivos

Objetivo General

Recolectar los datos generados minuto a minuto por los sensores para su posterior procesamiento mediante el Big Data

Objetivos Específicos

Diseñar e implementar la arquitectura de datos para recolección, procesamiento y almacenamiento.

Implementar el uso de DataWharehouse para la gestion de los datos.

Crear un dashboard para la capa de presentación en la página del proyecto

Metodología

Fase de Diseño

Maquetación de datos a nivel de modelo entidad relación, transformación en modelo multidimensional y sondeo para la implementación de PowerBI

Implementación

Insersión de datos padre, creación de jobs para insersión de lecturas, procesamiento ETL y carga de datos en DataWharehouse. Creación del tablero en PowerBi

Análisis

Procesamiento de datos y generación de informes analíticos.

Resultados y Conclusiones

Resultados Principales

  • Implementación exitosa del modelo multidimensional para el procesamiento de datos
  • Automatización de ETL para la carga de datos masiva
  • Desarrollo de DataWharehouse optimizado para consultas analíticas

Conclusiones Clave

  • El procesamiento Big Data permite un análisis eficiente de grandes volúmenes de datos
  • La implementación de un DataWharehouse facilita la gestión y análisis de datos históricos
  • Power BI demuestra ser una herramienta efectiva para la visualización de datos procesados
