Acerca del Proyecto

El proyecto SmartParks busca monitorear parques urbanos en tiempo real mediante sensores IoT conectados por WiFi, LTE o LoRaWAN.

Recolección de Datos

Los sensores recolectan datos como temperatura y cantidad de personas presentes.

Almacenamiento

Esta información es almacenada en una base de datos relacional para su posterior análisis.

Análisis

Procesamiento de datos para generar insights y métricas relevantes.

Toma de Decisiones

Facilitamos la gestión eficiente de espacios públicos basada en datos.

Objetivos

Objetivo General

Implementar un sistema de monitoreo inteligente para parques urbanos utilizando tecnologías IoT, que permita la recolección y análisis de datos en tiempo real para la toma de decisiones informada.

Objetivos Específicos

Diseñar una red de sensores IoT para la medición de variables ambientales y afluencia de visitantes.

Desarrollar una plataforma de almacenamiento y procesamiento de datos en tiempo real.

Crear dashboards interactivos para la visualización de métricas clave.

Metodología

Fase de Diseño

Planificación de la red de sensores y selección de tecnologías IoT apropiadas.

Implementación

Despliegue de sensores y desarrollo de la infraestructura de datos.

Análisis

Procesamiento de datos y generación de informes analíticos.

Resultados y Conclusiones

Resultados Principales

  • Implementación exitosa de red de sensores en 5 parques piloto
  • Recolección de más de 10,000 puntos de datos diarios
  • Precisión del 95% en mediciones ambientales

Conclusiones Clave

  • La tecnología IoT permite monitoreo efectivo de espacios públicos
  • Los datos en tiempo real mejoran la toma de decisiones
  • Se identificaron patrones de uso importantes para la gestión
